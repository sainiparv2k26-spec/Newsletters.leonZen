About

A Calgary newsletter space built for clarity and consistency.

This website is designed as a clean, static newsletter publication for YYC-focused writing. Visitors can read published pieces, while all new newsletters are added manually by editing the site code.

It is simple, fast, lightweight, and ideal for GitHub Pages hosting.

How publishing works

To publish a new newsletter, just edit the posts array in script.js, save, commit, and push to GitHub.
