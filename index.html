About

A minimal publishing site built for clarity.

This website is designed as a static publishing space for blogs, newsletters, and journal entries. Public visitors can only read posts. New content is added manually by editing a simple JavaScript array.

It is lightweight, GitHub Pages-friendly, and easy to maintain without any backend or dashboard.

Static site setup

No backend, no login, no admin panel. Just edit the post data in script.js, push to GitHub, and your updates go live.
